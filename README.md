# Honeybee - Apiary & Hive Management

Honeybee is a subdomain and module within the **Rural Utility Cost** ecosystem. It provides practical estimators and calculators for rural hive management, apiary planning, honey production forecasting, and beekeeping cost tracking.

## Ecosystem Context
This application is one of 25+ subdomains under the Rural Utility Cost (RUC) network. It shares a common UI framework, disclaimer policy, and branding standard to ensure a unified user experience across the entire suite of tools.

## Tech Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)

## Core Features
1. **Tool Hub**: An aggregated view of all beekeeping tools grouped by categories (harvest, season, equipment).
2. **Standardized Calculators**: Consistent, interactive calculators for estimating honey yields, swarm timings, and equipment costs.
3. **Cross-Network Portfolio**: Integrates dynamically with the master `portfolio-sites.md` reference to allow easy navigation to other Rural Utility Cost subdomains.
4. **Unified Disclaimer System**: Global compliance components to ensure disclaimers are correctly styled, positioned, and linked to the master domain's legal policy.

## Development Commands
- `npm install` - Install dependencies
- `npm run dev` - Start local development server
- `npm run build` - Build for production
- `npm run lint` - Run typechecking
