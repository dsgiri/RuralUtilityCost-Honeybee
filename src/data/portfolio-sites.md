## Update Instructions
- **Do not delete sites.** If a site is retired, change its Category to `Retired`.
- **Formatting:** Maintain the exact Markdown table format with 6 columns.
- **Parsing:** The application parser relies on pipes `|` to split the columns.
- **Syncing:** Whenever this file is updated on the main repository, it serves as the source of truth for all subdomains via local data loading.
